其实，安装这个lrzsz真的很简单，开一个随笔真是暴殄天物了，只需要

在超级用户下打一句命令：

	yum install lrzsz

或者，在普通用户打一句命令，需要输入超级用户密码：

	sudo yum install lrzsz
